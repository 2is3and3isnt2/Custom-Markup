How To Use

The first thing that you need to do is to link it in the <head> tag like so:

<link rel="stylesheet" type="text/css" href="https://custommarkup.ml/dist/main.min.css">

Or you can use NPMJS:

npm install copythat

Just experiment by using the tags in the list below. Just put in text as a child in the tag.

  • <c-r>
  • <c-g>
  • <c-b>
  • <c-o>
  • <c-y>
  • <c-i>
  • <c-v>
  • <bg-r>
  • <bg-g>
  • <bg-b>
  • <bg-o>
  • <bg-y>
  • <bg-i>
  • <bg-v>
  • <awesome>
  • <cool>
  • <nice>
  • <insane>
  • <beautiful>
  • <old>
  • <container>
